# 텔레그램 봇 만드는 방법
1. 텔레그램에서 BotFather 채팅방에 접속  
2. 채팅창에 /newbot 명령어를 입력하고 전송버튼  
3. 텔레그램 봇이름을 채팅창에 입력. (*봇 이름은 반드시 _bot으로 끝나야함)  
4. 한 번 더 봇 이름을 입력
